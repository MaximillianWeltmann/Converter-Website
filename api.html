API Dokümantasyonu

Quantum Dönüştürücü API'si ile kendi uygulamalarınızda dosya dönüştürme işlemlerini otomatikleştirin

Quantum Dönüştürücü API'si, geliştiricilerin kendi uygulamalarında dosya dönüştürme, sıkıştırma, yeniden boyutlandırma ve şifreleme işlemlerini programatik olarak gerçekleştirmesine olanak tanır. RESTful API mimarimiziyle istemci uygulamanızdan hızlı ve güvenilir dönüştürme işlemleri gerçekleştirebilirsiniz.

Başlarken

Quantum Dönüştürücü API'sini kullanmaya başlamak için aşağıdaki adımları izleyin:

  1. API anahtarınızı alın
  2. RESTful API son noktalarımızı kullanarak istekler gönderin
  3. İşlem sonuçlarını alın ve yanıtları işleyin

API'miz, tüm modern programlama dilleriyle uyumludur ve her istekte HTTP başlıklarında API anahtarınızı kullanmanızı gerektirir.

API Temel URL

https://api.quantum-donusturucu.com/v1

Kimlik Doğrulama

Tüm API isteklerinde kimlik doğrulama gereklidir. API anahtarınızı X-API-Key HTTP başlığında göndermeniz gerekir:

GET /v1/formats HTTP/1.1
Host: api.quantum-donusturucu.com
X-API-Key: your_api_key_here

API anahtarınızı güvenli tutun ve public istemci kodu içine yerleştirmeyin. Backend'den API çağrıları yapmanızı öneririz.

Hız Limitleri

API kullanımında abonelik planınıza bağlı olarak aşağıdaki hız limitleri geçerlidir:

Plan İstek Limiti Maksimum Dosya Boyutu
Ücretsiz 100 istek/gün 25 MB
Temel 1,000 istek/gün 100 MB
Pro 5,000 istek/gün 500 MB
Enterprise Sınırsız 2 GB

Limit aşımlarında API 429 Too Many Requests durum kodu döndürür.

Hata Yönetimi

API hataları standart HTTP durum kodları ve JSON yanıt formatı ile iletilir:

{
  "error": {
    "code": "invalid_api_key",
    "message": "Geçersiz veya eksik API anahtarı",
    "status": 401
  }
}

Genel hata kodları:

HTTP Durum Hata Kodu Açıklama
400 bad_request İstek parametreleri geçersiz
401 invalid_api_key Geçersiz veya eksik API anahtarı
403 forbidden Bu işlem için yetkiniz yok
404 not_found İstenen kaynak bulunamadı
415 unsupported_format Desteklenmeyen dosya formatı
429 rate_limit_exceeded Hız limiti aşıldı
500 server_error Sunucu hatası

Dosya Dönüştürme

Dosya Yükleme ve Dönüştürme

POST /v1/convert HTTP/1.1
Host: api.quantum-donusturucu.com
X-API-Key: your_api_key_here
Content-Type: multipart/form-data

file=@dosya.docx
target_format=pdf
quality=high

İstek Parametreleri

Parametre Tür Gerekli Açıklama
file File Evet Dönüştürülecek dosya
target_format String Evet Hedef dosya formatı (pdf, docx, jpg, vb.)
quality String Hayır Çıktı kalitesi (low, medium, high). Varsayılan: medium

Yanıt

{
  "task_id": "a1b2c3d4e5f6",
  "status": "processing",
  "estimated_time": 5,
  "result_url": null
}

Dönüştürme Durumu Kontrol Etme

GET /v1/convert/a1b2c3d4e5f6 HTTP/1.1
Host: api.quantum-donusturucu.com
X-API-Key: your_api_key_here

Yanıt

{
  "task_id": "a1b2c3d4e5f6",
  "status": "completed",
  "result_url": "https://cdn.quantum-donusturucu.com/results/a1b2c3d4e5f6.pdf",
  "expires_at": "2023-09-22T12:00:00Z"
}

Sıkıştırma

Dosyaları sıkıştırmak için API son noktaları:

POST /v1/compress HTTP/1.1
Host: api.quantum-donusturucu.com
X-API-Key: your_api_key_here
Content-Type: multipart/form-data

file=@image.jpg
compression_level=75

İstek Parametreleri

Parametre Tür Gerekli Açıklama
file File Evet Sıkıştırılacak dosya
compression_level Integer Hayır Sıkıştırma seviyesi (1-100). Varsayılan: 80

Webhook'lar

Uzun süren dönüştürme işlemleri için webhook'ları kullanabilirsiniz:

POST /v1/convert HTTP/1.1
Host: api.quantum-donusturucu.com
X-API-Key: your_api_key_here
Content-Type: multipart/form-data

file=@large_video.mp4
target_format=avi
webhook_url=https://your-app.com/webhook

İşlem tamamlandığında, belirtilen webhook URL'sine bir POST isteği gönderilir:

{
  "task_id": "a1b2c3d4e5f6",
  "status": "completed",
  "result_url": "https://cdn.quantum-donusturucu.com/results/a1b2c3d4e5f6.avi",
  "expires_at": "2023-09-22T12:00:00Z"
}
